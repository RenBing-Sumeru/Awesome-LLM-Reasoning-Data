Evidence contract

Every answer tells you where the claim comes from.

Ask the Atlas first checks the companion paper, then this repository's paper maps and cards, and only then uses model background knowledge when repository evidence is limited.

  1. 01 Companion paper Highest-trust explanation when the primer covers the concept.
  2. 02 Repository atlas README, guides, tracks, cards, and generated metadata.
  3. 03 Model background Clearly labeled fallback for in-scope gaps.
01 Ask inside the field or repo

Questions can cover post-training reasoning data, verifiers, rewards, agent trajectories, evaluation, audit topics, or how to navigate this project.

02 Ground in atlas sources

Answers retrieve README, guides, paper maps, cards, generated metadata, and the configured companion paper text.

03 Turn curiosity into growth

GitHub login enables quota, star/fork rewards, feedback, and admin analytics that reveal missing docs and cards.
