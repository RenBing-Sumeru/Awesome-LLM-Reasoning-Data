The method begins with agent trajectories drawn from the paper's released data resources. Agent-RRM is trained to produce an analysis, critique, and score for each trajectory, so that one model output contains both explanatory text and a reward-like signal.

Three consumers are then studied. Reagent-C uses the critique to revise a trajectory, Reagent-R uses the score as reward guidance, and Reagent-U uses both channels. The experiment therefore tests not only whether the feedback can be generated, but also which way of consuming the same record helps an agent.
