ScaleSWE combines large-scale environment reconstruction, test generation, problem writing, and teacher trajectories into one production pipeline for real pull requests. Existing SWE benchmarks often preserve issues, patches, and tests, but this work emphasizes automated conversion into training episodes at scale.

Multi-agent construction and executable tests are not guarantees of semantic fidelity. The novelty is the scalable link between realistic repository changes and agent optimization, which still requires testing, provenance, and human audit of the generated task statement.
