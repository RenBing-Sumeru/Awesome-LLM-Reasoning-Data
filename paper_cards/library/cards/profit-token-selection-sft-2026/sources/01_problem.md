Single-reference SFT forces a model to imitate every token of one answer even though many surface expressions are interchangeable. Collecting multiple valid references can reduce that overfitting, but it multiplies data construction and training cost. Sample-level selection cannot distinguish the logical core of a response from stylistic or low-information segments inside the same response.

This paper asks whether one reference answer can be turned into a more selective supervision signal without building a multi-answer dataset. The training object is an instruction-response pair together with the current model's probability for every reference token. The decision is which token losses are allowed to update the SFT model; it is not a claim that low-probability text is globally incorrect or that probability alone verifies an answer.
