FOT treats difficulty as model-relative rather than as a fixed property such as length or topic. A sample is harder when the base model assigns it higher cross-entropy loss and when its token pattern appears less covered by the estimated pre-training distribution. Sorting the full training set by this joint score approximates a curriculum that starts near the base model's familiar distribution and gradually moves toward the target domain.

The curriculum is deliberately temporary. The first epoch uses easy-to-hard ordering, with random shuffling only inside contiguous score blocks; later epochs randomly shuffle the whole dataset to avoid overfitting to one sequence. Thus the proposal does not replace training data or add a teacher label. It changes the consumer schedule of the same records and argues that early ordering prepares the model for later difficult updates while later randomness preserves ordinary optimization behavior.
