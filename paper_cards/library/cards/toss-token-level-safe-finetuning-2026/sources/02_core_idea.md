TOSS changes the training record from an undifferentiated instruction-response pair into the same pair plus a per-token loss mask. The text need not be deleted from the dataset, and the model can still read the response prefix during teacher forcing, but masked target positions contribute no supervised gradient. This makes the unit of curation a token-level learning event rather than an entire prompt or response.

Its selection signal is the loss difference between two reference models that represent competing desiderata: preserving safe behavior and retaining task utility. Tokens that look especially favored by the safety-degraded model relative to the utility reference are ranked as higher risk and masked globally. TOSS-Pro then refines the harmful reference with selected high-risk examples, making the risk model progressively more tailored to the harmful behavior the method seeks to suppress.
