Use this method when preference data are fixed and quality varies over the course of DPO, but generating new data or running a costly selection model is impractical. It is a small integration point for a DPO-style trainer: retain the pairs, expose the current margin, and schedule the focal exponent. It is less suitable when the main problem is label corruption, domain coverage, or cultural representation, because it changes pair influence rather than repairing or diversifying the underlying records.
