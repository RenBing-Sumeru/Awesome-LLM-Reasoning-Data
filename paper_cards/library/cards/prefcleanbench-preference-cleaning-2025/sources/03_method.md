The benchmark starts with Anthropic-HH, UltraFeedback, PKU-SafeRLHF, and HelpSteer2. It implements thirteen cleaning approaches: LLM judges, reward-gap and reward-model-voting approaches, and prompt- or difficulty-based heuristics. Each approach marks questionable pairs and produces variants by removing records or, where applicable, flipping preference labels at specified cleaning fractions.

For each original and cleaned variant, the protocol first performs SFT on prompts and chosen answers, then runs preference optimization. The main setting uses Llama-3-8B and DPO; the released code also supports CPO, AOT, KTO, IPO, SLiC, rDPO, and ORPO. It compares model outputs using win-tie rate and held-out average reward, then repeats analyses across backbones and objectives to test generalization.
