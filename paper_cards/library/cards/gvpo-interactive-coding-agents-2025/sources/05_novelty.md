Prior-work baseline: Terminal-only GRPO-style rewards reveal little about where an interactive coding trajectory failed and provide no deterministic intermediate feedback. What changes is GVPO combines group policy optimization with outcome- and process-verifiable signals for agents acting in interactive coding environments. This is a current direction signal because the verifier is central to the data object, not an incidental evaluation metric.
