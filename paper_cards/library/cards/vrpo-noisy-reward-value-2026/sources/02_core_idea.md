VRPO treats a value model as a semantic noise regulator. A variational information bottleneck compresses the rollout representation so that the value prediction retains return-relevant information while discarding irrelevant detail. This is paired with auxiliary losses derived from a frozen language model's entropy and perplexity, directing the value model toward semantically uncertain or difficult tokens rather than merely fitting noisy returns.

The resulting value estimates enter advantage computation and therefore change what the policy is reinforced to do. The paper's key distinction is that it does not assume noisy rewards can be repaired at their source; it uses denser value supervision to compensate for their downstream effect. High-entropy and high-perplexity token selection makes this adjustment targeted rather than applying identical semantic loss to every token.
