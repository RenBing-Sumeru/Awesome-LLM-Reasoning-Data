The information-bottleneck value model maps a rollout state to a Gaussian latent representation and predicts return from that latent. Its loss combines return prediction with a Kullback-Leibler penalty that limits information retained from the input. Separately, a frozen language-model head identifies tokens above entropy and perplexity thresholds; entropy and negative log-probability losses on those subsets regularize the value model's internal representation. The complete value loss is then used inside a PPO-style optimization loop.

The paper tests learned dialogue rewards and rule-based majority-vote rewards. In rule-based training it uses five samples for voting, with 32 used in test-time optimization; it evaluates math and science with Qwen and Llama-family variants, and dialogue under a learned reward model. Reproduction requires reward-noise construction, rollout and voting protocol, value latent architecture, token thresholds, auxiliary-loss weights, policy initialization, and evaluation judge versions, because each alters the resulting advantage targets.
