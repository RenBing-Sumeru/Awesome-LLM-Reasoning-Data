The consumed object remains an ordinary instruction-response subset; the intervention is the rule that chooses that subset. This framing matters because it prevents a selection method from receiving credit merely for producing attractive examples on inspection. A useful selector must improve the outcome of a matched SFT run relative to repeated random choices drawn from the same pool, with the same subset size, backbone, number of epochs, and evaluation suite.

The authors' core proposal is an evidence standard for using selectors, rather than a new optimization objective. They test LESS, IFD, SelectIT, entropy, diversity, ZIP, and token length alongside random sampling, report variability across repeated choices, and apply significance tests. Their conclusion is practical: at the evaluated scale, simple diversity or length baselines can be at least as informative as elaborate self-scoring pipelines, so a curator should demonstrate a stable gain before declaring a method better than random.
