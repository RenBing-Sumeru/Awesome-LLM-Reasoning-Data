s1’s contribution is a deliberately simple, open baseline for test-time scaling: a small curated trace set plus explicit budget control, evaluated as a scaling curve. CoT distillation and length control already exist; the new signal is to put them in a transparent matched recipe rather than attribute gains to an opaque search system.

The approach does not prove that continued generation reliably fixes errors. Extra tokens can also extend an incorrect path, so the mechanism must be evaluated with controlled budgets and isolated data.
