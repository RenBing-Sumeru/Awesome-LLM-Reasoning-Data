The method uses Monte Carlo Tree Search to create branching reasoning paths instead of treating independently sampled completions as flat examples. Each node is an intermediate natural-language step, and search statistics expose how that step contributes within a trajectory. An external mathematics tool then checks the step itself, supplying an executable local signal rather than a stylistic judgment.

GroundedPRM fuses tool-verified step correctness with outcome feedback into a hybrid reward and backpropagates it through the tree. It converts the resulting trace into a generative record containing binary correctness and a rationale. The reward model is consequently trained to produce both a verdict and an explanation from a problem and its intermediate reasoning sequence.
