The core record is a tool-use context paired with a chosen and rejected trajectory. It retains tool specifications, calls, observations, and the comparison label, so a reward model can learn not merely whether an answer sounds good but whether the action sequence is operationally appropriate.

The official release is named ToolPref-Pairwise-30K. Its hosted configuration currently exposes more rows than the name suggests, so a reuse study should pin the exact split and configuration rather than treating the title as a complete count specification.
