Data Whisperer uses in-context learning as a proxy for fine-tuning influence. A candidate example appears among a small group of demonstrations, the model predicts answers for separate queries, and the measured accuracy or ROUGE score is initially credited to every demonstration in that context. Repeating these trials lets every record accumulate an empirical demonstration score without updating any model weights.

The key refinement is to distribute credit by attention. The method reads a selected layer's attention from the first prediction token to each demonstration, normalizes for example length, and weights the trial score accordingly. A record is therefore selected not only because it appeared in a successful few-shot context, but because the model attended to it while answering those queries; averaged weighted scores determine the final top-k subset.
