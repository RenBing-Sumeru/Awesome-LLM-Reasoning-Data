The method estimates a local slope matrix of cross-domain effects. Starting from the current model, it temporarily trains for a few steps on each fine-tuning dataset alone, restores the model and optimizer state, and measures how every target and constrained evaluation changes. A negative slope means beneficial transfer for an evaluation domain; a positive slope signals interference or potential forgetting.

At each update, the slope matrix becomes a short-horizon linear prediction of each evaluation loss under candidate mixture weights. The optimizer minimizes predicted target losses while penalizing a constrained domain if its predicted loss rises above the base-model reference. The resulting weights are used only until the next scheduled probe, so the mixture can first emphasize target data and later shift toward auxiliary data that regularizes preservation.
