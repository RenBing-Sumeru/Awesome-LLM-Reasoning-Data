Specializing an LLM through SFT can improve a target domain while eroding safety, factual knowledge, commonsense reasoning, or instruction following. Fixed mixtures of target and auxiliary data cannot react when transfer turns into interference during training, and a grid search over mixtures becomes expensive as the number of available sources grows. Existing adaptive mixers may change weights, but they do not directly encode a requirement that protected capabilities remain above a reference level.

DynaMiCS asks how to decide the next source-level SFT mixture when some evaluation domains must improve and others must not degrade. Its data object is a batch sampled from one of several named fine-tuning datasets; its evaluation object can be a held-out loss or a non-differentiable metric on a constrained domain. The contribution is a policy for consuming those source datasets under explicit performance constraints, not a new benchmark or a release of a new general-purpose corpus.
