TPO 以反事实方式定义一个 token 的视觉相关性。相同的提示和前序回答 token 被评估两次：一次使用原图，一次使用加入噪声的扰动图。原始 logit 的差异就是视觉锚定分数。如果扰动图像会显著改变一个 token 的预测，该 token 就被视为依赖视觉证据，而不是仅仅依赖语言先验。

该分数随后被自校准为偏好对两侧相反的奖励行为。在入选回答中，视觉锚定 token 获得更大奖励；在拒绝回答中，校准反转，使视觉锚定的问题 token 获得更低奖励。因此，该方法不是在优化前一次性过滤 token。它使每个 token 对类 DPO 目标的贡献具有动态性并依赖模型，将训练信号绑定到视觉敏感性。
