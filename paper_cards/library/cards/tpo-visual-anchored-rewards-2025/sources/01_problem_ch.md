视觉语言偏好优化常为整个入选或拒绝回答分配一个奖励。然而，幻觉通常是局部的：少数名词、属性或关系可能没有图像支持，而周围文本没有问题。现有细粒度方法依赖昂贵的人类片段标签、固定的幻觉奖励或单独的评分模型。它们难以直接显示，在当前训练步骤中哪些生成 token 真正以视觉输入为依据。

论文要回答的是：既有多模态偏好对如何在不增加新的细粒度标注时产生可扩展的 token 级落地奖励。它测量当原图被扰动图替换时，每个 token 输出 logit 的变化。较大变化标记一个视觉锚定 token。训练目标随后根据该 token 出现在入选还是拒绝回答中，对它进行不同的奖励或惩罚，并随策略变化更新该信号。
