The contribution is to make visual dependence, rather than response membership alone, determine a token's training reward. Previous preference pairs say which response is better but not which words within it must be grounded in the image. TPO produces that missing granularity from a controlled perturbation of the same input image, avoiding a manually annotated segment reward and avoiding a separate token scorer.

Its self-calibration is also distinctive. Positive and negative responses do not receive the same visual score with opposite labels; the calibration changes the direction of reward so that a visually dependent token has an appropriate role on each side of the pair. Because the policy-side score is recomputed throughout training, the record's effective reward surface evolves with the model. This is a dynamic training-use rule for fixed pairs, not a new hallucination dataset.
