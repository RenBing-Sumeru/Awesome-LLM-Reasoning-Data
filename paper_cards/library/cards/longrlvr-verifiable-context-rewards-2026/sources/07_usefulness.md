Use this work when training a model must improve both evidence localization and final answering over a fixed long context. Starting from long documents, a generator, and a defensible evidence annotation scheme, reuse the record format and context-reward manager to create an RL dataset, then compare answer-only RLVR and the combined reward under the same base model and rollout budget. Do not transplant the F-score rule to tasks without trustworthy reference evidence; first measure whether alternative correct chunk sets are common.
