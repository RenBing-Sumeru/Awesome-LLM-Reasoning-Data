The method assumes that a reference set of necessary chunks is available and that chunk-level F-score reflects useful grounding; incomplete references can punish valid alternative evidence or reward a superficially matching selection. Its 46K records are generated and quality-scored by Qwen3-235B-A22B, so reuse should separately audit generator/judge bias, document licenses, and overlap with evaluation corpora. The reported results cover controlled long-document QA, not an agent that retrieves from a changing external corpus.
