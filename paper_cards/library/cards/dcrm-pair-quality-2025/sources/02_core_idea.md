DCRM defines a high-quality preference pair as one with a substantial reward-relevant margin but little extraneous difference. It uses the reward-model margin as a proxy for the difference the model should learn, then calibrates that margin by token-level edit distance and the reference policy's log-probability difference. A high score therefore favors pairs that differ clearly in judged quality without differing unnecessarily in other ways.

The paper then applies this score to pair construction rather than merely reporting it as an analysis metric. Given several responses to a prompt, Best-of-N-squared pairing evaluates candidate pairs and keeps the one with the highest DCRM score. This makes response-pair selection model-relative: the same texts can receive different scores when the reference policy assigns them different probabilities.
