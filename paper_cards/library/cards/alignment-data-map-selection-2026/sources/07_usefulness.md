Use this paper when a preference collection has multiple candidate responses per prompt or when a large labeled corpus is too costly to train on wholesale. Compute and retain the individual scores, mean, variance, region assignment, correlation, scorer identity, and thresholds alongside the original record; these are necessary to explain a selection decision or reconsider it when the judge changes. It is useful for prioritizing annotation and training, but it should not be used as an automatic deletion rule without checking whether low-score regions contain valuable edge cases.
