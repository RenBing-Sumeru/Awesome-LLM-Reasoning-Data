Static proportional or uniform mixtures choose source weights before training, whereas several earlier dynamic approaches rely on gradient similarity, uncertainty estimation, actor networks, proxy models, or validation supervision. DYNAMIX SFT changes the object being optimized from a fixed recipe to a time-indexed source distribution. It derives feedback by asking how much a one-step probe on a source reduces the current model loss, then uses that feedback to change later SFT batches.

The distinctive mechanism is not simply using a bandit or computing a loss reduction. It combines a look-ahead reward, an exponentially smoothed estimate, a prior-scaled Boltzmann policy, and a uniform floor so adaptation cannot collapse the source mixture. The ablation on the prior is especially relevant: it tests the claim that dynamic exploitation should be anchored to the original coverage rather than replacing that coverage with an unconstrained reward ranking.
