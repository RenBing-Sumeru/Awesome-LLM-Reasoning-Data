The new object is a scored hypothesis, rather than an unfiltered teacher answer. Conventional output distillation asks a student to copy a response; ReDis records the candidate rule, tests its consequences on demonstrations, and uses the resulting ranking both to remove weak supervised targets and to define an alignment preference.

This creates an auditable bridge between synthetic rationale generation and model tuning: the same local fitness estimate governs selection and pair construction. It is not a formal verifier, because a language model performs rule following, and it is not a general solution to all in-context learning. Before reuse, inspect the satisfaction threshold, score-gap rule, teacher prompt, and whether a target domain admits comparable demonstration checks.
