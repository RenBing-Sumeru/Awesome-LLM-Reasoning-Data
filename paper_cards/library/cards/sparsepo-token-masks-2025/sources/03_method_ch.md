SparsePO 从一个 token 级偏好表述出发，为每个 token 插入一个介于 epsilon 和一之间的掩码值。奖励掩码乘以入选和拒绝回答中策略、参考模型的对数比；KL 掩码乘以下一个 token 的 KL 散度。得到的目标比较被掩码的奖励总和，并减去被掩码 KL 项之间的差异。论文允许两种掩码相同或分别学习，因此奖励强调和参考正则化不必遵循相同的位置。

对于基于激活的掩码，该方法对参考模型各层的前馈、残差和注意力激活进行标准化并取平均。对于可学习掩码，小型 ReLU 网络读取每层隐藏状态，最终投影将它们合并；ReLU 自动诱导稀疏值。掩码与偏好优化一起训练，并在训练后被忽略。实验使用合成情感偏好、Anthropic HH 对话偏好和 Reddit TL;DR 摘要偏好，并比较稠密、共同稀疏和独立稀疏配置。
