传统偏好优化将入选回答或拒绝回答中的每个 token 都视为同样有用。这可能注入大量微弱或带噪的监督，尽管一小组极端 token 就可能承载了大部分偏好信号。已有的选择性方法往往依赖昂贵的细粒度标签或复杂的 token 选择过程，因此难以应用于普通的成对偏好数据。

论文要回答的是：不收集新的 token 标注，如何识别真正驱动既有回答级偏好的 token。它提出先在目标数据的一个抽样子集上训练一个适中的 DPO oracle，再使用 oracle 与参考模型的概率比作为 token 级奖励估计，并只用选出的极端 token 训练目标策略。因此，决策不再是是否保留一个回答对，而是该回答对的哪些部分应当影响训练目标。
