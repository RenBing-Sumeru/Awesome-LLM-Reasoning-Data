The new contribution is a record-level transformation from response pair to token-masked pair. Rather than asking a reward model to label every action or training on all response text indiscriminately, SePO derives a reusable token score from an ordinary DPO oracle and uses it to decide which tokens become the actual learning surface. This changes the supervision granularity of existing preference data without requiring a new human annotation channel.

The work also separates weak supervision from weak generation. A smaller oracle need not produce a better response than the target policy; it only needs to rank which positions carry useful preference evidence. That distinction supports the reported weak-to-strong setting and distinguishes the method from a direct distillation pipeline. The paper's random-mask control makes the key claim falsifiable: the gain is attributed to the estimated reward ranking, not simply to training on fewer tokens.
