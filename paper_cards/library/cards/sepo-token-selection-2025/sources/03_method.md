The procedure has three stages. First, a reference model is obtained through supervised fine-tuning on chosen responses from a moderate random subset. DPO on that subset produces an oracle model. For every token in the full target preference set, SePO computes the log probability of the oracle divided by the reference, conditioned on the prompt and preceding tokens. The paper argues that this score estimates a decomposed token-level reward under its stated assumptions.

Second, within each chosen response SePO retains the top k percent scoring tokens, and within each rejected response it retains the bottom k percent. Third, it optimizes a length-normalized contrastive objective using only these masked token likelihoods. The main experiments use 30 percent selection. The authors also test weak-to-strong variants: a small oracle selects tokens for a stronger policy, or a high-quality oracle selects useful positions from weak, out-of-distribution preference data.
