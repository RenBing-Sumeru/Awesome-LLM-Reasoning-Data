The underlying record remains a question, a long reasoning trace, and a final answer from the OpenR1-Math pool. The contribution does not claim that every selected trace is a newly verified proof; it changes the decision rule that determines which existing records become supervision.

The rule combines two signals. Difficulty is defined relative to the base model, while trace length is measured after exact duplicate reasoning steps are removed. Ranking by both signals favors examples that are challenging for the chosen student and contain more than repeated text.
