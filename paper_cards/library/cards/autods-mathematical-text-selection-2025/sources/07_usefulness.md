Use this work when building a documented mathematical-text mixture for continual pretraining and when a full human-labeling program is infeasible. Start from the released score and source mixture, select a band under a fixed token budget, and compare it with uniform, length-based, and domain-specific alternatives using the same base model and schedule. Treat the score as a curation signal rather than a solution-correctness label, and preserve the chosen prompt and model version so that another curator can reproduce the decision.
