Main evidence: reports budgeted gains over best-of-N and FLOPs-matched comparisons where smaller models with extra TTC can beat larger models on suitable prompts.

Metrics and comparisons should be read together with the budget. For this track, the important question is not only whether accuracy improves, but which resource changed: data examples, generated rollouts, verifier calls, search branches, RL updates, or base model size.

Evidence limits: depends on verifier calibration, task difficulty estimates, and benchmark overlap controls.
